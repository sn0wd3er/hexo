## 변수

값을 저장할 수 있는 공간

### 사용법

- DELCARE @[변수명] [타입] : 변수 선언
- SET @[변수명] = [값] : 변수 값 대입
- SELECT 변수명,변수명 : 변수 출력

![Variable](/img/sqlVariable.PNG)

- Decimal은 소수 값을 표현할 수 있다. `DECIMAL(전체 길이,소수점 길이)`

## 타입 변환

- CAST(변수명 AS 타입)
- CONVERT(타입,변수명)
